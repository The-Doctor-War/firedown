Browse — grouping subtitle tracks under their parent video

YouTube hands us 50+ caption tracks per video (manual + ASR + auto-translated). Flat emit buries the list under language soup. Four grouping treatments below, all using the same three videos: one with many captions, one with two, one with none. Subtitle chip view at the bottom shows what each treatment looks like when the user is filtering.

EN manual track EN with red tint = auto-generated (ASR) Captions counted include manual + ASR + auto-translated

Variant A — inline expandable

Parent video row carries a CC 52 badge. Tap the row to expand inline; indented sub-rows appear below. Default reveals the top few (system locale, English, original) with a "show all" tail.

Variant B — pill on parent, bottom sheet to pick

Parent row carries a compact CC 52 pill. Tap opens a bottom sheet listing every language with checkboxes, system locale and English pre-checked. Captions are not entities in the Browse list at all — they ride as a property of the parent until the user picks.
